About GWF Toolbox

Google Webfont Toolbox is a lightweight browser-based tool for previewing and refining Google Fonts before you use them in a project. It allows you to test font styles, and quickly grab the embed code or CSS needed to integrate your chosen fonts into a design.

It takes inspiration from the work of the Google Webfont Helper, creating a better designed system that allows you to see more information about the fonts in question. While the original project allowed you to download font files to self-host, this Tolbox goes a little further:

  • You can find out more data about the typefaces, including designer copyright and licensing information.
  • You can get a clearer indication of whether your chosen font is going to work by previewing all of the variants.
  • You can check whether your chosen font will work with your theme colours - and test them against WCAG AA and AAA standards
  • You can view the suitability of your chosen font for a design project, with x-height and cap height measurements

Fonts are loaded directly from the Google Fonts API, so you'll always be working with the latest available styles. You can also download TTF versions of the fonts (WOFF2 coming ... soon? Hopefully?)

Tech Stack

This project is built using a mix of lightweight, modern web technologies:

  • HTML5 and CSS3 for structure and design
  • JavaScript and jQuery for interactivity and DOM manipulation
  • Google Fonts API to pull font data and previews live
  • LocalStorage to remember user preferences between visits
  • Font Awesome for all of the pretty, awesome, and pretty awesome icons

It also utilises these amazing scripts, without which we couldn't have built most of the features here:

  • Prism.js for previewing the code
  • JSZip for saving your chosen fonts into a ZIP file
  • Filesaver.js for help writing and saving downloaded files to your system
  • opentype.js for help analysing font data, including typeface anatomy
